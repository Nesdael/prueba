# Student system, added, updated, removed
This project is a simple system that allows for the management of basic student information. The system allows for the registration, consultation, updating, and deletion of students, as well as the structured storage of information for later use.
## Requirements
- python 3.x

## functions
- register new students
- shows the students
- You can update any student information
- You can remove a student by their ID
- can look for a student

## What does the code do?
The code allows you to register students and save their information in a list stored in a Json file.
It can display all the student information stored un the list.
you can search for a student by ID or name, update their information, and delete a student usinf their ID
